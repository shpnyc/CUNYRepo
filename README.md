# CUNYRepo
# Repository of projects for CUNY
#Details regarding Project 1
1. The first project consists of an analysis of students' test performance. Utilising a method of descriptive statistics, data cleansing and visualization.
  a. The data set used in the first project was taken from the GitHub repository: https://github.com/CunyLaguardiaDataAnalytics/datasets/blob/master/2014-15_To_2016-17_School-_Level_NYC_Regents_Report_For_All_Variables.csv
  b. The data set is loaded into Pandas and a heads and tails analysis is performed to get an overview of the data rows. Commands used: head(), tail()
  c. The data set is summarised and analysed using descriptive statistics. 
    1a. Selected school compared to other schools in the district
    2a. Selected school compared to other schools in the borough
    3a. Selected school compared to other schools in the dataset
  d. The data analysis contains a cleaning task
  e. The data analysis contains one visualisation
  f. Summary of Analysis: The data file contains 212331 entries with 15 columns. There are two data types: int64 and object. The numeric data in the file had to be converted in order to use the group by function. Utilised the Scatter chart feature for visualisation.
